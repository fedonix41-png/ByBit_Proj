# План исправления ошибок импорта

## Анализ проблемы

**Ошибка:** `ImportError: cannot import name 'OptionalAuth' from 'app.core.deps'`

**Причина:** Несогласованность между файлами после множественных частичных изменений:
- `app/core/deps.py` содержит функцию `get_optional_user`
- `app/core/__init__.py` пытается импортировать `OptionalAuth` (старый класс)
- `server.py` использует `get_optional_user`

## Затронутые файлы

| Файл | Проблема |
|------|----------|
| `app/core/__init__.py` | Импортирует несуществующий `OptionalAuth` |
| `app/core/deps.py` | Содержит `get_optional_user`, но не экспортируется как раньше |
| `app/database/session.py` | Имеет `get_db`, которая конфликтует с deps.py |

## План исправления

### Шаг 1: Удалить дублирующий get_db из deps.py
`get_db` уже определён в `app/database/session.py` для FastAPI dependency injection.
deps.py должен импортировать его оттуда, а не переопределять.

### Шаг 2: Обновить app/core/__init__.py
Заменить:
```python
from .deps import (
    get_db,  # удалить - уже есть в session.py
    get_current_user,
    get_current_active_user,
    get_admin_user,
    OptionalAuth,  # заменить на get_optional_user
)
```

На:
```python
from .deps import (
    get_current_user,
    get_current_active_user,
    get_admin_user,
    get_optional_user,
)
```

### Шаг 3: Обновить __all__ в app/core/__init__.py
Заменить `OptionalAuth` на `get_optional_user`

### Шаг 4: Обновить app/core/deps.py
Добавить импорт `get_db` из session.py:
```python
from ..database.session import SessionLocal, get_db
```

### Шаг 5: Проверить server.py
Убедиться что импорты корректны (уже использует `get_optional_user`)

## Порядок действий

1. Исправить `app/core/__init__.py` - убрать `OptionalAuth`, добавить `get_optional_user`
2. Исправить `app/core/deps.py` - импортировать `get_db` из session.py, удалить дубликат
3. Пересобрать Docker образ
4. Протестировать

## Файловые изменения

### app/core/__init__.py (строки 26-32, 63)
```python
# Было:
from .deps import (
    get_db,
    get_current_user,
    get_current_active_user,
    get_admin_user,
    OptionalAuth,
)

# Должно быть:
from .deps import (
    get_current_user,
    get_current_active_user,
    get_admin_user,
    get_optional_user,
)
```

И в `__all__`:
```python
# Было:
"OptionalAuth",

# Должно быть:
"get_optional_user",
```

### app/core/deps.py (строки 7, 14-20)
```python
# Было:
from ..database.session import SessionLocal

def get_db() -> Session:
    """Get database session for FastAPI dependency injection."""
    db = SessionLocal()
    try:
        yield db
    finally:
        db.close()

# Должно быть:
from ..database.session import SessionLocal, get_db
# (удалить дублирующую функцию get_db)
```
