# Обзор проекта

## Назначение

**Bybit P2P Automation** — умный помощник (Smart P2P Arbitrage Assistant) для операторов P2P-торговли. Проект представляет собой единый шлюз-терминал, который связывает биржу **Bybit** и процессинговую платформу **Zavod**, автоматизируя рутину с помощью AI, но оставляя все ключевые решения за человеком.

Ключевые функции помощника:
- Автоматическое извлечение реквизитов из диалогов с контрагентами (AI)
- Интеллектуальная оценка рисков мошенничества (FraudAnalyzer)
- Связывание сделок Bybit с заявками на Zavod
- Обязательный человеческий контроль на критических этапах (Human-in-the-Loop)

## Целевая аудитория

- **P2P-операторы** → автоматизация копипаста реквизитов, чеков и типового общения
- Разработчики → интеграция AI (как помощника) в торговые системы
- Команды → auditable процесс обработки заявок и принятия решений

## Ключевые возможности

### AI-агенты
- **IntentClassifier** — анализ намерений пользователя в чате для навигации по сделке
- **ResponseGenerator** — помощник генерации типовых вежливых ответов и запросов реквизитов
- **PaymentParser** — OCR + AI для проверки чеков, приходящих от платформы Zavod
- **FraudAnalyzer** — проверка предоставленных клиентом реквизитов и паттернов общения на мошенничество

**Провайдеры:** OpenAI, Anthropic, Groq, Together, Mistral, Local (Ollama), Mock

### LangGraph оркестратор
- 12 узлов обработки
- Точки прерывания для подтверждений
- Персистентность состояния (SQLite)

### Telegram-бот
- Текст, голос (Whisper), фото (GPT-4 Vision)
- Интерактивное меню (InlineKeyboard)
- Контекст диалога

### Bybit API
- Testnet для безопасной разработки
- Mock-режим для тестирования
- Полная интеграция с P2P API

### Веб-интерфейс
- FastAPI + WebSocket
- REST API для мониторинга
- Real-time обновления состояния
- **Табовая навигация:** Объявления, Ордера, Чат, Аккаунт, Мониторинг
- **CRUD для объявлений:** создание, редактирование, удаление
- **Управление ордерами:** списки, детали, mark_as_paid, release_assets
- **Чат:** отправка сообщений, загрузка файлов
- **Информация:** баланс, аккаунт, контрагент

### База данных
- PostgreSQL + SQLAlchemy
- Alembic миграции
- История ордеров, сообщений, решений

## Безопасность

- **Testnet-first** — разработка только на тестовой сети
- **Human-in-the-Loop** — подтверждение критических действий
- **Mock-режим** — тестирование без реальных API
- **JWT аутентификация** — access + refresh токены
- **Rate limiting** — защита от abuse
- **Security headers** — XSS, CSRF, clickjacking защита
- **Audit logging** — аудит всех операций
- **DB backup** — ежедневное резервное копирование
